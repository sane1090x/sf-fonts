# Install Apple's San Francisco Fonts on Arch Linux

This script quickly installs Apple's San Francisco font family on Arch Linux. Although I haven't extensively tested it, I believe it's reliable. However, for security reasons, I recommend reviewing the code before running it.

To use the script, clone this repo and run it with `bash install-sf-fonts.sh`. It may also work on other systems.
